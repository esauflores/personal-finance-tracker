## Tech Stack

- **Framework:** React + TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **Charts:** Recharts
- **State Management:** Zustand
- **Storage:** localStorage
- **Linting:** ESLint + Prettier
